Hello there!

Open up the console to try out these methods

  • returns a list of all dog breeds
    Dog.every()
  • returns a random dog image (number argument is optional and returns that amount of random dog images)
    Dog.random(number)

    Ex: Dog.random(3)

  • returns an array of dog images of the dog breed passed in (breed argument is required. random argument is optional and returns a single random image of the selected breed)
    Dog.breed(breed, random)

    Ex: Dog.breed("hound", true)

  • returns an array of sub breeds of main breed passed in. (breed and subBreed argument are required. random argument is optional and returns a single random image of the selected subBreed)
    Dog.subBreed(breed, subBreed, random)

    Ex: Dog.subBreed("hound", "afghan", true)

  • returns an array of subBreeds for the passed in breed. (breed argument is required)
    Dog.subBreedList(breed)

    Ex: Dog.subBreedList("hound")
